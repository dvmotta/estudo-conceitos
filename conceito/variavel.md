# *Variáveis*
É um espaço reservado na memória utilizado para armazenar dados durante a execução de um programa. A variável é composta por 3 elementos básicos, sendo eles:  
&emsp;&emsp;**Nome:** O nome é um identificador utilizado no código para referenciar a variável.  
&emsp;&emsp;**Tipo:** Define o tipo de informação armazenada, como texto, números ou valores booleanos, além das operações permitidas sobre ela.  
&emsp;&emsp;**Valor:** É o dado armazenado pela variável. Geralmente é atribuído durante sua criação, mas pode ser alterado ao longo da execução do programa.
## Tipos de varáveis
### Simples
&emsp;&emsp;**Inteiro (int):** Representa números inteiros, positivos, negativos ou zero.  
&emsp;&emsp;**Booleano (bool):** Representa valores lógicos, podendo assumir apenas dois estados: *verdadeiro (True)* ou *falso (False)*.  
&emsp;&emsp;**Ponto flutuante (float/double):** Representa números reais, ou seja, números que possuem casas decimais. Em algumas linguagens existem os tipos *float* e *double*, sendo o *double* mais preciso. Em Python, utiliza-se apenas *float*, que internamente possui precisão equivalente à do *double* de outras linguagens.  
&emsp;&emsp;**Texto (str):** Representa uma sequência de caracteres, podendo conter letras, números, símbolos ou até mesmo um único caractere.
### Composto
&emsp;&emsp;**Lista (list):** É uma coleção ordenada e mutável de elementos, em que cada elemento possui uma posição (*index*).  
&emsp;&emsp;**Tupla (tuple):** Assim como a lista, é uma coleção de elementos, porém a tupla é *imutável*, geralmente usada para guardar informações que não podem mudar ao longo do programa.  
&emsp;&emsp;**Dicionário (dict):** Atua como coleção de elementos, porém organiza os dados por *chave e valor*. Assim como os dicionários físiscos, é necessário procurar a *chave* para conseguir o *valor*.  
&emsp;&emsp;**Conjuntos (set):** É uma coleção de elementos não ordenada e que não permite duplicações, ou seja, ela não possui posições (*index*).
