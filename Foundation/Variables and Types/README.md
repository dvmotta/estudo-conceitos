# *Variables*
A variable is a space in memory used to store data while a program is running. It has three basic elements:  
&emsp;&emsp;**Name:** The name used to refer to the variable.  
&emsp;&emsp;**Type:** Defines the type of data stored, such as text, numbers, or boolean values and determines which operations are allowed.  
&emsp;&emsp;**Value:** The data stored in the variable. It is usually assigned when the variable is created, but it can be changed during program execution.

## Variable types
### Primitive types
&emsp;&emsp;**Integer (int):** Represents whole numbers, which can be positive, negative, or zero.  
&emsp;&emsp;**Boolean (bool):** Represents logical values, which can be either `True` or `False`.  
&emsp;&emsp;**Floating-point (float/double):** Represents real numbers with decimal places. Some programming languages have *float* and *double*, with *double* providing higher precision. In Python, only *float* exists, and its precision is comparable to *double* in many other languages.  
&emsp;&emsp;**String (str):** Represents a character sequence and may contain letters, numbers, symbols, or even a single character.
### Compound types
&emsp;&emsp;**List:** An ordered collection of elements that can be modified during program execution. Each element has a position called an *index*.  
&emsp;&emsp;**Tuple:** Similar to a list, but it is immutable, commonly used to store data that should not change during program execution.  
&emsp;&emsp;**Dictionary (dict):** A dictionary is also a collection of elements, but it stores data as *key-value* pairs. Like a dictionary, you use a *key* to find its associated *value*.  
&emsp;&emsp;**Set:** An unordered collection of unique elements. Sets do not allow duplicates and do not use indexes.
