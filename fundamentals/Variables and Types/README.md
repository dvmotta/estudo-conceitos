# *Variables*
A variable is a space in memory used to store data while a program is running. It has three basic elements:  
* **Name:** The name used to refer to the variable.  
* **Type:** Defines the type of data stored, such as text, numbers, or boolean values and determines which operations are allowed.  
* **Value:** The data stored in the variable. It is usually assigned when the variable is created, but it can be mutated during program execution.

## Variable types
### Primitive types
* **Integer (`int`):** Represents whole numbers, which can be positive, negative, or zero.  
* **Boolean (`bool`):** Represents logical values, which can be either `True` or `False`.  
* **Floating-point (`float`/`double`):** Represents real numbers with decimal places. Some programming languages have *float* and *double*, with *double* providing higher precision. In Python, only *float* exists, and its precision is comparable to *double* in many other languages.  
* **String (`str`):** Represents a sequence of characters and may contain letters, numbers, symbols, or even a single character.
### Compound types
* **List:** An ordered collection of elements that can be modified during program execution. Each element has a position called an *index*.  
* **Tuple:** Similar to a list, but it is immutable, commonly used to store data that should not change during program execution.  
* **Dictionary (dict):** A dictionary is also a collection of elements, but it stores data as *key-value* pairs. Similar to a real dictionary, you use a *key* to find its associated *value*.  
* **Set:** An unordered collection of unique elements. Sets do not allow duplicate values and do not use indexes.
